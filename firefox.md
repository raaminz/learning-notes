[[tools]]

| key                | meaning           |
| ------------------ | ----------------- |
| Ctrl + ]           | Forward           |
| Ctrl + w           | close tab         |
| Ctrl + Shift + w   | close window      |
| Ctrl + T           | new tab           |
| Ctrl + N           | new window        |
| Ctrl + tab         | go to right tab   |
| Ctrl + shift + tab | go to left tab    |
| Alt + D            | focus address bar |
| Ctrl + K           | focus search bar  |
|                    |                   |
